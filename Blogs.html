Steganography

Cybersecurity and encryption technologies have always interested me.

This image of an African river was taken by a family member and has not previously been published. It contains not only a hidden message, but also the complete transcript of Shakespeare's play "Hamlet".

To decode it, you will need to download an open source terminal app called "Steghide", following the steps below:

  1. To install steghide on Debian-based systems, type sudo apt-get install steghide
  2. Download the image to your computer without compressing it.
  3. Open the terminal and navigate to the place you want to extract the file.
  4. Copy in the following text to your terminal, push space, but DO NOT hit enter: steghide extract -sf
  5. Drag and drop the downloaded image location to the terminal, and push space
  6. Add the following text to the end of the text: -p mikerocks
  7. The complete command adjusted for the filepath on your PC should look like this:
    steghide extract -sf '/home/flash/Desktop/Hidden-text.JPG' -p mikerocks
  8. Hit enter then open the freshly extracted text file from the pwd location.

For more information on how to use this tool, check out this external post
